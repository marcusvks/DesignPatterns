Explicação: 
O Padrão de Design 'Adapter'
Tem por objetivo adaptar o funcionamento de uma funcionalidade existente para um comportamento novo.
Causando pouco impacto na aplicação, apenas trocando sua injeção de dependencia para o adaptador.


Necessidade / Problema: 
Alterar o comportamento de alguma funcionalidade da aplicação para um novo comportamento causando pouco impacto.
